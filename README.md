### About Project

다년간의 사용 경험으로부터 만들어낸 가장 완벽한 가계부, Scrooge <br>

Scrooge는 단순히 수입과 지출이 완벽하게 똑같아야 차액이 생기지 않는 가계부가 아닙니다. <br>

스스로 예산을 정해 계획적인 지출을 할 수 있도록 도와주는 웹 앱입니다.<br>
